Twitter API Home Page

Instructions to run this page:

  • In the following page, you will have the opportunity to interact with five different RESTful API created by gathering data from a given JSON file Source here
  • To interact with the first three APIs, the user simply clicks on the 'Get...' button and the results will be automatically reloaded without refreshing the page via AJAX calls
  • For the last two APIs, it requires an input for the user which are either tweet ID or user's screen name, respectively. It is recommended to play around with the APIs mentioned above to gather important information for the API request as the parameter is mandatory; otherwise, the results will be null
  • Lastly, after interaction with each of the API, the link to the endpoint is displayed within the section. This way, the user can directly see the JSON file where the results come from.
